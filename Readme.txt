== Description ==

Note: Added Plugin Options.

About the Author Box plugin displays the author bio and website link at the starting of the posts without disturbing the design.  You can see a live example on <a href="http://devilsbackyard.com/post/wordpress-about-the-author-box-plugin/">the plugin page.</a>
If you run a multi author blog, then this is plugin can be very beneficial. It is a highly customizable plugin (from colours to position).

For more information on customizing, visit <a href="http://devilsbackyard.com/post/wordpress-about-the-author-box-plugin/">the plugin page.</a>
 
== Installation ==

1. Upload `about-the-author-box.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now see your posts for the box!
